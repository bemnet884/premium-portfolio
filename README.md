# Bemnet | Premium Portfolio

A minimalist, high-performance one-page developer portfolio built for speed, refined typography, and a sophisticated aesthetic. Designed specifically to showcase full-stack, mobile, and 3D web expertise.

## 🚀 Key Features

- **Blazing Fast Performance**: Built natively with Vanilla Javascript and HTML via Vite. Zero heavy framework overhead to guarantee instantaneous load speeds and perfect Lighthouse scores.
- **3D Interactive Hero**: Leverages **Three.js** WebGL to render an elegant, cursor-tracking 3D wireframe that immediately visually validates 3D expertise.
- **Case Study Modals**: A fully custom, glassmorphic `<dialog>` component dynamically hydrates with deep-dive technical challenges and architectural decisions when exploring projects.
- **Graphical Process Timeline**: Features a highly responsive desktop-to-mobile alternating Zigzag timeline tracing the specific software engineering lifecycle.
- **Horizontal Project Gallery**: Custom JavaScript implementation utilizing continuous `requestAnimationFrame` loops for butter-smooth horizontal scrolling with touch/hover pause mechanics.

## 🛠 Tech Stack
- HTML5 / CSS3 variables & layout algorithms.
- Vanilla Javascript (`IntersectionObserver`, `requestAnimationFrame`).
- **Three.js** (WebGL 3D engine).
- **Vite** (Hyper-fast local development and production bundling).

## 💻 Local Development

1. Setup the project and install the Three.js dependency:
   ```bash
   npm install
   ```

2. Start the hot-reloading development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
